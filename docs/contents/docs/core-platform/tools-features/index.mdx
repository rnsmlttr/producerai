# Tools & Features

## Slash Commands

Quickly trigger specific actions in the chat interface.

| Command | Action | Example |
| :--- | :--- | :--- |
| `/create` | Skip conversation, generate instantly | `/create jazz piano lo-fi` |
| `/remix` | Modify the selected track | `/remix add reverb` |
| `/stems` | Separate audio components | `Exports WAVs` |

## Core Capabilities

<CardGrid>
  <Card title="Stem Separation" icon="layers">
