# Songwriting Guide

The most common mistake is trying to generate a full 3-minute song in one prompt. The best results come from a **component-based workflow**.

<Step>
  <StepItem title="The Chorus">
    Start with the hook. It's the most reliable way to establish the theme.
  </StepItem>
  <StepItem title="The Verse (Extend Backwards)">
    Use `/extend` and select "Prepend". This generates a buildup leading into your chorus.
  </StepItem>
  <StepItem title="The Outro">
    Extend forward to resolve the track.
  </StepItem>
</Step>
